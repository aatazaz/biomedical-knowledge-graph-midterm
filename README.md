# Biomedical Knowledge Graph

## Overview
This project builds a biomedical knowledge graph using Neo4j and Python.

## Features
- Drug-Disease relationships
- Gene-Disease associations
- Graph visualization
- Automated data pipeline

## How to Run
1. Start Neo4j
2. Run:
   python scripts/neo4j_import.py
   python scripts/analysis.py

## Results
- 60 Nodes
- 40 Relationships
